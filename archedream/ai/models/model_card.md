# Model Card — ArcheDream Insight Assistant

Overview
- Purpose: бережные юнгианские инсайты, символы, вопросы, практики.
- Inputs: dream text/audio (STT), mood, affects, associations.
- Outputs: summary, archetypes(≤3), symbols, questions, practice, safety notes.

Intended Use
- Саморефлексия; не медицинская; 18+; этический режим по умолчанию.

Data
- Weak supervision + экспертная правка; мульти‑язычная; приватный корпус.
- Минимизация PII, удаление по запросу, агрегированные профили.

Safety
- Фильтры триггеров/NSFW/PII; отказ от диагнозов; кризисные ресурсы.
- Explainability: evidence highlighting; uncertainty language.

Limitations
- Возможные культурные смещения; не интерпретирует редкие идиолекты.
