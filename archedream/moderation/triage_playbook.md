# Плейбук модерации — триаж

Категории
- A: немедленная опасность (самоповреждение, насилие) → ресурсы помощи.
- B: нарушение правил (агрессия, доксинг) → скрыть, предупредить, бан по шкале.
- C: чувствительный контент → пометка, триггер‑фильтры.

Шаги
- Оценка риска → действие → запись в журнал → уведомление автора.
