# Этический кодекс

Область применения
- Продукт, ИИ‑инсайты, сообщество, консультации, обработка данных.

Основы
- Не медицинское приложение; не заменяет психотерапию; не диагностируем.
- 18+; кризисные ресурсы по гео; предупреждения о триггерах.
- Согласия прозрачные и гранулярные; опция полного локального режима.

Приватность и безопасность
- Локальная обработка по умолчанию; облако — только с явным согласием.
- E2EE для чатов; хранение снапшотов архетипов без сырого текста при желании.
- Экспорт/удаление по требованию; минимизация данных и сроков хранения.

ИИ‑прозрачность
- Объяснимость: подсветка фраз ↔ архетипов; избегаем категоричности.
- Анти‑предвзятость: регулярные аудиты, жалобы, исправление.
- Запрет на медицинские трактовки и опасные советы.

Сообщество
- Анонимность по умолчанию; запрет на доксинг и агрессию.
- Многоуровневая модерация; «не мой триггер» фильтры для ленты.

Консультации
- Проверка квалификации аналитиков; прозрачные цены и SLA возвратов.
