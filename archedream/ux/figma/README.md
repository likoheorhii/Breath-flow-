# Figma импорта и прототипирование

1) Установите плагин Tokens Studio.
2) Импортируйте `ux/tokens/figma_tokens.json` (Create styles from tokens).
3) Создайте страницу «Wireframes», сетка: 12 кол, gutter 16, margin 16.
4) Перетащите SVG из `ux/export/svg/` на канву; установите автолэйаут для секций.
5) Задайте стили: текст = Body/Display, заливки = tokens.colors.*.
6) Прототип: свяжите «Записать сон → Инсайт → Интеграция», колесо → колесо.
7) Экспортируйте интерактивный прототип (Present) и поделитесь ссылкой.
