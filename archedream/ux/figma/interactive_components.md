# Interactive Components — руководство

Компоненты
- Tab (active: true|false) — клик меняет состояние (Change To)
- Button (variant: primary|ghost)
- Practice (done: true|false) — клик переключает done

Как использовать
1) Перетащите вариант из ассет‑пэнела на кадр
2) В Prototype поставьте «Smart Animate 220ms» для переходов по Change To
3) Для Tab: соедините Tab‑бар с соответствующими кадрами (Home/Record/Insight/Integrate)
4) Для Practice: клик по карточке → Change To done=true, затем модалка «рефлексия» (по желанию)
