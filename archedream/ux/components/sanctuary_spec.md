# «Святилище» — UX‑спека

Цель
- Снижение возбуждения, фокус на внутреннем опыте, минимальный шум.

Визуал и звук
- Тёмный индиго фон, мягкие градиенты, микропульсации звёзд.
- Аудио‑тишина по умолчанию; опциональный «дыхательный тон» 60–72 BPM.

Взаимодействие
- Таймер 10–20 мин «тихого режима» (DND); haptics low.
- Отложенные уведомления; мягкие переходы 200–250 мс.

Доступность
- Режим «без анимаций»; крупные интерактивы; контраст ≥ 4.5:1.
