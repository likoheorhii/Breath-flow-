# Компоненты (каталог)

- Topbar: заголовок, левая/правая зона (иконки), тёмный фон.
- Tabs: 4 кнопки, состояние active/hover, контраст ≥4.5.
- FAB: круглая кнопка, primary.
- Card: контейнер с радиусом 16, варианты surface/brand.
- Button: primary/ghost/danger, pill‑радиус.
- Input: textarea, select, helper text, ошибки.
- Chips: evidence/теги, многострочные, сворачиваемые.
- Modal: центр, backdrop, esc/кнопка закрытия.
- Wheel: SVG контейнер 300×300, сегменты 12, легенда.
- Practice item: название, длительность, чек «сделано», опцион. заметка.
