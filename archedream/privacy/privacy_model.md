# Приватность

Модель данных
- dreams(text, mood, affects, symbols) — локально; облако только с согласием.
- archetype_profiles(vector) — можно хранить без сырых снов.
- messages — E2EE; сервер видит метаданные минимум.

По умолчанию
- Локальная обработка; публичных публикаций — нет без явного действия.
- Экспорт/удаление: в настройках, мгновенно, с подтверждением.

Сроки хранения
- Сырые сны в облаке (если включено) — 12 мес; профили — 24 мес.

Контроль
- Гранулярные согласия; «святилище» — временный офлайн‑режим.
